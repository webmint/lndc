# lndc-test

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### To run for development with backend
```
npm run start
```

### Compiles and minifies for production
```
npm run build
```

### To run production with backend
```
npm run start-prod
```
### To view production use
```
http://localhost:4000/
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### Run your unit tests
```
npm run test:unit
```
